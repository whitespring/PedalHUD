## [1.0.0] - 2026-03-10

### Added

- Initial public release of PedalHUD.
- macOS host app for Bluetooth/device setup and virtual camera activation.
- CoreMediaIO camera extension that composites live cycling metrics over a webcam feed.
- Shared core package for metrics, overlay configuration, and app-group persistence.
- Sparkle update support through GitHub Releases appcast metadata.

### Documentation

- Public README, development setup guide, architecture guide, contribution guide, and security policy.

### Release

- Signed, notarized GitHub release assets and public Sparkle feed.
